Detailed Description

Class for direct access to the pixels of a Bitmap.

Note
After structural modifications of a Bitmap (e.g. changes of the size or the PixelFormat) all corresponding PixelAccessors become invalid and may no longer be used.
A PixelAccessor keeps a Reference to the associated Bitmap. If there exists no other reference to the Bitmap, it will be deleted if the PixelAccessor is deleted.
