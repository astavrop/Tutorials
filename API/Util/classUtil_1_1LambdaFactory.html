Detailed Description

template<class ObjectType, typename IdentifierType, template< class, typename > class FallbackPolicy = FallbackPolicies::ExceptionFallback>
class Util::LambdaFactory< ObjectType, IdentifierType, FallbackPolicy >

Generic factory for objects allowing lambdas with variable number of arguments as creator functions.

The factory template can be instaniated to generate different kinds of factories. The generated factory generates objects of type ObjectType.

Template Parameters
ObjectTypeBase type for all objects that are generated by the factory
IdentifierTypeType of the identifier that specifies which object creator to call
FallbackPolicyTemplate with a function onUnknownType() that handles the case that the requested object type was not found
