Detailed Description

A high level wrapper for a network connection of two endpoints (using tcp). It allows synchronizing of key-value pairs and sending of small data packages using multiple virtual channels. These channels allow multiple interleaved communications using a single network connection.

Note
Internally creates an additional thread.
Is not designed for sending huge blocks of data at once.
The maximum length of a key is 256 characters - additional characters are truncated.
The given TCPConnection must only be accessed by the DataConnection or the data will get corrupted! [DataConnection]
