Detailed Description

Broadcasts data to a set of DataConnections.

Note
The DataConnections can at the same time used for other communications.
A closed DataConnection is automatically removed if it is closed.
Deleting a DataBroadcaster does not automatically close the assciated connections. [DataBroadcaster]
