Detailed Description

template<class InternalBaseType, class InternalDerivedType, class WrapperDerivedType>
class Util::PolymorphicWrapperCreator< InternalBaseType, InternalDerivedType, WrapperDerivedType >

Generic creator for wrapper objects.

This functor takes an object of type InternalBaseType. By using RTTI, it checks if the object is of subtype InternalDerivedType. If this is the case, it returns a new wrapper of type WrapperDerivedType with the object as constructor parameter.

Note
Only use this creator for polymorphic types InternalBaseType and InternalDerivedType.
Template Parameters
InternalBaseTypeBase type of internal objects that are given to the creator
InternalDerivedTypeConcrete type of internal objects for which a wrapper shall be created
WrapperDerivedTypeConcrete type of wrapper object that will be created
