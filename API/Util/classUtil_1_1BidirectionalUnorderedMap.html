Detailed Description

template<class Key_Left, class Key_Right, class Hash_Left = std::hash<Key_Left>, class Hash_Right = std::hash<Key_Right>, class Pred_Left = std::equal_to<Key_Left>, class Pred_Right = std::equal_to<Key_Right>>
class Util::BidirectionalUnorderedMap< Key_Left, Key_Right, Hash_Left, Hash_Right, Pred_Left, Pred_Right >

class for storing bidirectional mappings between objects internaly uses two unordered maps for left to right and right to left mapping all iterators are const_iterators to ensure that both internal maps stay consistent
