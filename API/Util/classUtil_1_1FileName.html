modifies the filename such that it has the given ending, examples: "file" –> "file.ending" "file.xy" –> "file.ending" "file.ab.xy" –> "file.ab.ending" "" –> ".ending" ".xy" –> ".ending"

Parameters
endingthe new ending to be used for this filename can be specified either as ".ending" or as "ending"
Note
if ending is specified as ".xy.ab", only ".ab" is used
