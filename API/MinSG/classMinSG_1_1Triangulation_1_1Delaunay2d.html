Detailed Description

template<typename Point_t>
class MinSG::Triangulation::Delaunay2d< Point_t >

Delaunay2d The implementation is based on the Java applet '2D Delaunay Triangulation Java Applet' by Luke Sunder Parige (parig.nosp@m.e@ew.nosp@m.s.uiu.nosp@m.c.ed.nosp@m.u) dated August 10, 1999.

See Also
http://members.fortunecity.com/lukesunder/dt2d.htm
