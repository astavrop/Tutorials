Detailed Description

template<typename _T>
class Rendering::MeshUtils::Simplification::UpperTriangularMatrix< _T >

Storage of an upper triangular n-by-n square matrix. Only the entries in the upper triangle and the diagonal are stored and no memory is used for the entries in the lower triangle.
