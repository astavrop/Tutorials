Detailed Description

Simple usage example for GUI.

The example uses a GUI::GUI_Manager to create a GUI::Window, a GUI::Textfield, and a GUI::Button. Then, it registers a GUI::ActionListener to wait for a click onto the button. When the button is clicked, the text field is cleared.
